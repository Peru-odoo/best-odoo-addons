Website Enhanced Google Tag Manager

This module enables e-commerce conversion tracking and remarketing with Google Tag Manager.

Features:

  • Add Google Tag Manager ID to website configuration.
  • Integrate your sales with Google Analytics.

Installation:

  1. Clone this repository.
  2. Add this directory to your addons path (e.g. --addons-path=addons,path/to/this/repo).
  3. Install the module website_enhanced_google_tag_manager.
  4. Modify the new fields on the website configuration page.
